# Mind Companion

AI-Powered Mental Wellness Companion

Production-ready mobile app with LangGraph multi-agent orchestration, grounded RAG, Flutter frontend.

**Disclaimer**: This is NOT a therapist, doctor, or diagnostic tool. It provides psychoeducation and evidence-based coping support. Always seek professional help when needed.

## Features
- LangGraph multi-agent workflow
- Structured Knowledge Base (CBT, ACT, DBT, etc.)
- Mood Tracking, Journaling, Voice Chat
- Weekly Insights
- Secure, scalable architecture

## Tech Stack
Backend: FastAPI, LangGraph, PostgreSQL, Redis, Qdrant
Frontend: Flutter, Riverpod
DevOps: Docker

See full documentation in the repo.

## Quick Start
```bash
docker-compose up --build
```
